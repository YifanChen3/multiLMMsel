The `MORES` package contains the implementation of the estimation and variable selection of moment-based high-dimensional linear mixed effects model. The detail of this method will be available soon at arXiv. 

To install this package from GitHub, type in R console

`devtools::install_github("YifanChen3/MORES", build_vignettes = TRUE)`

Note that this installation requires R package `devtools` (which can be installed using `install.packages("devtools")`).

Please check the accompanying vignette on how to use the `MORES` package. To read vignette, after the installation of `MORES` package, type in R console
`browseVignettes("MORES")`.
